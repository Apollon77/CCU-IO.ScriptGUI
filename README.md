# CCU-IO.ScriptGUI
## Version 0.22     !!! Noch nicht nutzbar !!!

CCU-IO.ScriptGUI ist ein Visueller Script builder mit dem es möglich ist Scripte für CCU.IO zu erstellen, zu bearbeiten und zu testen.
